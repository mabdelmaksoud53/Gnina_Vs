# High-Throughput Virtual Screening using GNINA

- This repository contains a workflow for performing virtual screening of ligand libraries against the **MAO-B** (Monoamine Oxidase B) receptor using **GNINA**, a deep learning-based molecular docking tool.

## Repository Structure

```bash
Gnina_Vs/
├── ligands/
├── resources/
├── screenshots/
├── README.md
├── Gnina_Vs.ipynb
└── LICENSE                 
```

- This workflow is optimized for `**Google Colab**` but can be adapted for local Linux machines.

### Installation

1. Clone this repository:

```bash
git clone https://github.com/mabdelmaksoud53/Gnina_Vs
cd Gnina_Vs
```
